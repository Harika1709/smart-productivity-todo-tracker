# Smart Productivity Todo Tracker 🚀

A modern and responsive productivity tracking web application built using HTML, CSS, and JavaScript.

## ✨ Features

- ✅ Add / Edit / Delete Tasks
- ♻️ Daily Recurring Tasks
- 📅 Date-wise Task Scheduling
- 📊 Productivity Dashboard
- 🌙 Dark / Light Theme
- 🔍 Search & Filter Tasks
- 📈 Weekly & Monthly Analytics
- 🎯 Productivity Score System
- 🎉 Confetti Celebration Effects
- 💾 LocalStorage Data Persistence
- 📱 Fully Responsive UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API
- Font Awesome
- Canvas Confetti

## 🧠 Key Functionalities

### Task Management
- Add tasks with category and priority
- Edit and delete tasks
- Mark tasks as completed

### Recurring Tasks
- Daily repeating tasks automatically regenerate each day

### Productivity Dashboard
- Today's progress
- Weekly completion chart
- Monthly statistics
- Productivity scoring

### Theme System
- Toggle between dark and light themes

---

## 📦 Storage

All tasks are stored using browser LocalStorage.
